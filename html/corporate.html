Corporate Landing Page

Year: 2023

This was a final project for an intermediate web development course. The students were given a Photoshop file of a website layout to copy and the goal was to code the layout as closely to the design as possible. The layout featured centering text on a background image, three-column designs, animations, and a mega-footer. This assignment also taught students the basics of CSS grid and how to split the webpage into 12 equal parts in the grid. I received a 95 on this final project.
